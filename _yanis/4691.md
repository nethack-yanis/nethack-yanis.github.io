In variants with both menuglyphs and object materials, the glyph shown in the
menu should match the object's material.

This could be done relatively easily with a hacky solution in ASCII windowports,
but a proper implementation for tiles-based windowports would be challenging
since it amounts to adding new tiles for every object-material combination.
